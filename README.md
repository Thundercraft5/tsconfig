TSConfigs for myself.

* NodeJS

	* `node12` (ESM, ES2019)
	* `node14` (ESM, ES2020)
	* `node16` (ESM, ES2021)

* Browser (All AMD)
    * `browser`
    * `browser-es5` (ES5)
    * `browser-es6` (ES6)
	* `browser-esnext` (ESNext)
* `base` (Base configuration for all others)
* `esm` (Base configuration for ESM-based configurations)
* `esm-node` (Same as `esm`, but for NodeJS)
